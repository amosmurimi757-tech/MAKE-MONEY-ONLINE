🚀 QUICK START (5 MINUTES TO LIVE)

Step 1: Create GitHub Repository
→ Go to github.com
→ Create new repository: "yourusername.github.io" (replace yourusername)
→ Make it PUBLIC
→ Click Create

Step 2: Upload Files
→ Click "Upload files"
→ Drag all files from this folder
→ Commit changes

Step 3: Enable Pages
→ Settings → Pages
→ Select "main" branch
→ Select "/" (root) folder
→ Click Save

Step 4: Wait
→ Wait 5 minutes
→ Visit https://yourusername.github.io
→ ✅ Your website is LIVE!
